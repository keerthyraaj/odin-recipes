# Odin Recipes

## Description
Odin Recipes is a basic recipe website built as part of The Odin Project's Foundations curriculum. The purpose of this project is to demonstrate and solidify my understanding of fundamental HTML concepts before moving on to CSS and styling. 

The site consists of a main index page that acts as a directory, linking out to various recipe pages. Each recipe page contains structured content including headings, paragraphs, images, unordered lists for ingredients, and ordered lists for step-by-step instructions.

## Skills Demonstrated
By completing this project, I have successfully practiced and demonstrated the following skills:
*   **Git & GitHub Workflow:** Creating repositories, cloning to a local environment, committing changes with meaningful messages, and pushing code to a remote repository.
*   **HTML Boilerplate:** Setting up valid HTML document structures using `<!DOCTYPE html>`, `<html>`, `<head>`, and `<body>` tags.
*   **Text Formatting:** Using structural headings (`<h1>` through `<h3>`) and paragraphs (`<p>`) to organize written content logically.
*   **Lists:** Implementing unordered lists (`<ul>`) for recipe ingredients and ordered lists (`<ol>`) for sequential cooking steps.
*   **Links & Navigation:** Utilizing anchor tags (`<a>`) to navigate between local files using relative paths, including deep-linking into subdirectories and navigating back to the home page using `../index.html`.
*   **Media Integration:** Embedding images using the `<img>` tag with appropriate `alt` descriptions for web accessibility.
